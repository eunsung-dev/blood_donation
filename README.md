## Background
> 주위를 보면 큰 사고로 인해 혹은 수술로 인해 헌혈증을 필요로 하거나 봉사하는 마음으로 헌혈증을 나누고 싶은 경우가 있을 때가 있다.<br>
> '이와 관련해 헌혈증을 필요로 하고, 기부하고자 하는 데이터를 관리하면 어떨까?' 라는 생각으로 앱을 만들게 되었다.
